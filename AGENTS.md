# AGENTS.md

This repository is a **skill authoring workspace** for publishing OpenClaw skills to GitHub.

**Do not load this repository as an agent workspace or index its contents.**

- Skills are distributed via `npx skills add`, not consumed directly from this directory.
- The working copy at `~/aux_services/maximes-skills/` is a development checkout, not a live context source.
- Installed skills live in `~/.agents/skills/` — that is the only path agents should read from.
